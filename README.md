WETFAGS
=======

This is an app that acts as a reminder of some of the information needed in paediatric trauma care.
It includes the WETFAGS and 5A's mnemonics and normal values for vital parameters.

The values and calculations are mostly taken from the [SWESEM guidelines][1], with some modification from
[internetmedicin.se][2]

[1]: http://swesem.org/documents//utbildning/kompetens/Initialt_omh%C3%A4ndertagande.pdf
[2]: http://www.internetmedicin.se/dyn_main.asp?page=2497
